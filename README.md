### 패드
- 마우스패드가 아니라 `트랙패드` 라고한다.
- 두 손가락으로 위아래로 굴리면, 스크롤
- 세 손가락으로 누르면, 화면생성
- 세 손가락으로 좌우로 굴리면, 화면이동
